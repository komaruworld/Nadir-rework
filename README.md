# Nadir-rework

A maintained fork of [Nadir](https://github.com/kohy-creates/Nadir) with updates and improvements.

## About this project

This project continues development of the original Nadir datapack, which is no longer maintained. It includes bug fixes, new features, and updates for modern Minecraft versions.
